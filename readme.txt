BBC World Service global news in your own language, as a widget in your site's sidebar.

== Description ==
The BBC World Service widget can be added to WordPress theme that supports widgets (most do). It is also available for a number of other platforms, including iGoogle, Adobe Air, Mac Dashboard, Vista Sidebar, Opera Widgets and Netvibes. To find out more, click the '**+**' button in the widget, or visit the [BBC World Service Widget home page](http://www.bbc.co.uk/worldservice/programmes/000000_widget_terms.shtml).

The BBC World Service Widget is available in various languages including - Turkish, Spanish, Russian, Portuguese (Brasil), Vietnamese, Hindi, Indonesia, Bangla, Arabic, Persian, Urdu, Chinese, UKChina, Learning English, Swahili, Somali, Nepali and Hausa, French (Afrique). 

WordPress is available in [many languages](http://codex.wordpress.org/WordPress_in_Your_Language). When the widget is first installed, it will attempt to find BBC content to match the language of your WordPress installation (it checks the [`WPLANG`](http://codex.wordpress.org/Installing_WordPress_in_Your_Language) property in your [`wp-config.php`](http://codex.wordpress.org/Editing_wp-config.php) file).

== Installation ==
1. [Download the plugin](http://downloads.wordpress.org/plugin/bbc-world-service-widget.zip) to your computer and unzip it
1. Upload the full `/bbc-world-service-widget` plugin folder to the `/wp-content/plugins/` directory of your WordPress site
1. Activate the plugin through your ['Plugins'](http://codex.wordpress.org/Managing_Plugins) admin panel
1. Add the widget to a sidebar in the ['Widgets'](http://codex.wordpress.org/Appearance_Widgets_SubPanel) admin panel
1. Edit the widget language in the admin panel, if required
